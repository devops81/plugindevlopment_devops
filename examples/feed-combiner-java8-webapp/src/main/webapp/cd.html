Continous integration/continous deployment

What we do under contnous integration

As a part of continous Integration we make the software component is build for each of the change performed. We make sure the required test cases are well integrated with the build system and post build reqired notification are sent across to respective users with build status based on that further decision could be taken. Below are few of the benefit of integratin g CI system

  • it runs the unit test cases automatically and publishing the reports
  • our CI system works as a information radiator - it gives all this information automatically
  • Its capable of doing automated deployments as per the release process on different environment
